About

A web-based application that analyses and visualizes data related to adverse events of drugs retrieved from open-source health-related databases.

Guidelines to select parameters for generating chart

  1. Select a Drug:

    Begin by selecting a specific drug of interest. Start typing the drug name, and our system will suggest options based on available data. Choose the drug you want to analyze. If you don't have a specific drug in mind, you can leave this field empty to explore adverse events by drug indication.

  2. Select an Adverse Event (Optional):

    If you want to focus on a particular adverse event associated with the selected drug, you can specify it here. The system will provide suggestions as you type. If you want to view all adverse events related to the drug, leave this field empty.

  3. Select a Time Frame (Optional):

    To analyze adverse events within a specific time frame, provide a start date and/or an end date. The system will retrieve data for adverse events reported between these dates. If you don't provide dates, the system will use default values, typically starting from 2004 and ending with the current date.

  4. Choose a Chart Type:

    Select the type of chart you want to generate: Bar Chart, Histogram, or Pie Chart. Each chart type provides a different way of visualizing the data, so choose the one that best suits your needs.

  5. Generate the Chart:

    Once you've made your selections, click the "Generate Chart" button. The system will process your choices and retrieve the relevant data from the FDA database.

  6. Interpreting the Chart:

    After the chart is generated, you can interact with it to gain insights. Hover over data points to view details, zoom in/out, or click on chart elements for more information.

  7. Explore Further (Optional):

    If you want to explore additional data or create a new chart, you can return to the parameter selection section and make different choices. You can refine your analysis by selecting different drugs, adverse events, or time frames.

  8. Export Chart (Optional):

    If you wish to save or share the chart, look for export options provided by the charting tool. You might be able to download the chart image or share a link.

  9. Data Availability:

    Keep in mind that data availability can vary. If you encounter a message saying "no data" after generating a chart, it may mean that there is no information available for the specific parameters you selected.

  10. Help and Support:

    If you have any questions or encounter issues while using the tool, refer to the provided help resources or contact our support team for assistance.
